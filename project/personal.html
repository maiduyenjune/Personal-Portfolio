Overview

This project involves creating my own portfolio website and branding, which includes designing website interfaces, web development, and incorporating appealing animations for added features.

Category

Personal Project

Tools

Figma, Visual Studio Code, After effect, Photoshop, Illustrator

Duration

6 weeks

Deliver

Design Process

I began by creating several logo concepts for your personal branding portfolio, drawing inspiration from my personality, preferences, photo, and job title. Refine these concepts to create a final logo and experiment with different color schemes.

Next, I brainstormed ideas that align with insights gained from the logo creation process.

I used Figma to design the website, incorporating ideas and leveraging After Effects to add engaging motion elements for a fun user experience.

And Final result..

Then I proceed to web development, coding the design using HTML, CSS, and JavaScript to bring your website to life.
